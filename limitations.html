⚠️ What TruffleScan Can't Do
🔸 CEX transfers — if tokens were sent from a centralized exchange, the trail stops there. We can't trace inside Coinbase, Binance, etc.
🔸 Bridges — cross-chain transfers break the hop trail. A token bridged from ETH to SOL will show the bridge as the origin.
🔸 Multiple senders — if a wallet received a token from multiple sources, only the earliest sender is traced.
🔸 Current balances — scan amounts reflect the first recorded transfer, not current holdings.
🔸 Very new tokens — tokens with limited on-chain history may return incomplete results.
🔸 API rate limits — during high traffic, some scans may be slower or require a retry.
