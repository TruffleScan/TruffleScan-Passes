🔍 Reading a Scan
TruffleScan calculates your real cost basis by pulling every historical buy and sell transaction for a wallet + token pair. This is not an estimate — it's the actual math from on-chain data.

Total Invested — how much you spent buying the token
Total Received — how much you got back from sells
Remaining Value — current value of tokens still held
Net PnL — your real profit or loss
🕵️ Reading an Investigation
Investigate traces a token backwards through wallets, hop by hop, until it finds the origin — a CEX, DEX, launchpad, or unknown source.

Origin Wallet — where the token started its journey
Hop 1, 2, 3... — intermediate wallets the token passed through
Investigated Wallet 🎯 — the wallet you submitted

Amounts shown are how many tokens each wallet received in its first recorded transfer — not current balance.
🛡 Reading a TruffleCheck
TruffleCheck runs 3 layers of security analysis:

Layer 1 — GoPlus flags (honeypot, mintable, blacklist, etc.)
Layer 2 — On-chain analysis (liquidity, holder distribution, top wallets)
Layer 3 — 24h and 48h behavioral checks (volume, price action, sell pressure)
